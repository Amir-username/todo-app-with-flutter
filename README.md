
# Todo App

A todo app with flutter and Isar database


## Installation

I just add the lib directory and pubspec.yml file, you should copy and replace them to your project root.

### dependencies:

```text
  isar: ^2.5.0
  isar_flutter_libs: ^2.5.0
  path_provider:
  universal_io: ^2.2.0
  shared_preferences: ^2.0.0
```
